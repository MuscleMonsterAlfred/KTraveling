# 云服务器选择
## 服务器类型对比

g6的特点如下：
- 依托神龙架构，将大量虚拟化功能卸载到专用硬件，降低虚拟化开销，提供稳定可预期的超高性能
- 计算：
  - 处理器与内存配比为1:4
  - 处理器：2.5 GHz主频的Intel ® Xeon ® Platinum 8269CY（Cascade Lake），睿频3.2 GHz，计算性能稳定
  - 支持开启或关闭超线程配置
- 存储：
  - I/O优化实例
  - 支持ESSD云盘、SSD云盘和高效云盘
- 网络：
  - 支持IPv6
  - 超高网络PPS收发包能力
## 通用型云服务价格对比
