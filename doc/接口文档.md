
## 1. 发送短信验证码

#### 1. 接口描述    

用户点击发送验证码后请求该接口

#### 2. 请求地址  

`ktravel/traveluserinfo/sms`  

#### 3. 请求方式  

**GET**  

#### 4. 请求参数  

Params参数

| 参数名       | 必选 | 类型      | 示例         |
| ------------ | ---- | ---------------- | ------------ |
| phone_number|    是   |   string     | +447536252189 |


#### 5. 返回示例 

{
    "msg": "success",
    "code": 0
}
